# Mohamed Ibrahim - Portfolio 🚀

A modern and responsive personal portfolio website built with Next.js, React, Tailwind CSS, Framer Motion, and Three.js.


---

## ✨ Features

- Modern responsive design
- Dark mode support
- Animated 3D background
- Smooth animations using Framer Motion
- Fully responsive for mobile, tablet and desktop
- Projects showcase
- Skills section
- Contact section
- SEO optimized

---

## 🛠️ Technologies

- Next.js
- React.js
- TypeScript
- Tailwind CSS
- Framer Motion
- React Icons
- React Three Fiber
- Three.js
